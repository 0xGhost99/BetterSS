## 📌 Descrizione

**BetterSS** è la versione **gratuita** del sistema ScreenShare per server PvP Minecraft 1.19+.
Fornisce tutte le funzionalità base per gestire le sessioni SS in modo professionale.

Vuoi funzionalità avanzate come GUI, Staff Mode, Vanish, AntiCheat integrato e molto altro?
👉 **BetterSS+ è in arrivo! Stay tuned.** 🔜

---

## ✨ Funzionalità Free

- 🌍 Mondo dedicato `betterss` automatico
- 🧊 Freeze avanzato
- 🚫 Anti-relog con ban automatico
- 📋 Scoreboard live
- 💬 Messaggi automatici
- ⏱️ Timer massimo SS
- 📁 Log YAML
- 🔔 Webhook Discord
- ⚙️ Config personalizzabile

## 🚀 Installazione

1. Scarica l'ultima versione da [Releases](https://github.com/0xGhost99/BetterSS/releases)
2. Metti `BetterSS.jar` nella cartella `plugins/`
3. Riavvia il server
4. Configura `plugins/BetterSS/config.yml`

---

## 🎮 Comandi

| Comando | Descrizione | Permesso |
|---------|-------------|----------|
| `/ss <player>` | Avvia SS | `betterss.use` |
| `/ss freeze <player>` | Freeza player | `betterss.admin` |
| `/ss unfreeze <player>` | Defreeза player | `betterss.admin` |
| `/ss clean <player>` | Termina SS | `betterss.admin` |
| `/ss ban <player>` | Banna player | `betterss.ban` |
| `/ss discord <player>` | Invia Discord | `betterss.admin` |
| `/ss status <player>` | Stato SS | `betterss.admin` |
| `/ss credits` | Credits | tutti |

## 🔐 Permessi

| Permesso | Descrizione | Default |
